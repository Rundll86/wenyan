# 文言

此项目为一个恶搞的**文言文**编程语言，其语法与文言文相似。

## 文件

代码文件以`.文言`为扩展名。

## 语法

所有语句末尾必须以句号结尾。

### 导入模块

```plain
《模块文件名》曰：导入符号1，...导入符号n。
```

### 调用函数

```plain
函数名，已知【参数1】为 值1，...【参数n】为 值n。
```

### 字面量

```plain
“字符串”
十一万四千五百一十四
```

### 定义函数

使用`求`关键字来返回。

```plain
涵义【函数名】，需知 参数类型1【参数1】，...参数类型n【参数n】：
    语句1。
    ...
    语句n。

    求 结果。
```
